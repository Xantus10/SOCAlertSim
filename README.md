# SOC Alert Sim

A React.JS frontend-only app for simulating simple alert triage in SOC.

This app basically just creates a pretty frontend around a JSON structure.

The JSON file can be passed in one of the following ways:

1. I have created a few presets hosted here on github
2. You may pass a URL to your own JSON file
3. Or you may upload a file from your local device

## Creating your own

The versatility of this app is that you can create your very own exercises! The only thing you need to handle is the correct JSON structure. For guides on this check out the JSON_guides folder of this repository.
