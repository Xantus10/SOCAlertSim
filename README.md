# SOC Alert Sim

A React.JS frontend-only app for simulating simple alert triage in SOC. You may access it [here](https://xantus10.github.io/SOCAlertSim/)

This app basically just creates a pretty frontend around a JSON structure.

The JSON file can be passed in one of the following ways:

1. I have created a few presets hosted here on github
2. You may pass a URL to your own JSON file
3. Or you may upload a file from your local device

## Creating your own

The versatility of this app is that you can create your very own exercises!

Check out the creator section of this app (Availible in menu). It features a lot of preset alerts to get you started.

Currently from the following topics:

 - Login
 - Web
 - EDR - Files
 - EDR - Execution
 - Network

And for any additional changes, you can edit the json directly. For the correct structure, check out the JSON_guides
